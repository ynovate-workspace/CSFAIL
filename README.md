## Installation
1. Clone this repository

    ```bash
    git clone https://github.com/ynovate-workspace/csfail.git
    ```
2. ```bash
    cd backend
    ```
3.  Use the package manager [npm](http://www.npmjs.com/) to install dependencies.

    ```bash
    npm install
    ```
4.  ```bash
    cd ../frontend
    npm install
    ```
